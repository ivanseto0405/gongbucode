Welcome to the Beginner course. In this part of course, we
will start with understanding Hangeul(Korean Alphabet) and Korean Number. It is
really important to understand about hangeul and how to read it before starting
to read Korean Word.

To start the course you can see the modules menu in the left
side of the page or you can select from the table of content.

Hopefully we can finish this beginner part before jumping
into the next level, so good luck!
